# Frilans – React + TypeScript + Vite

![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)

Ett modernt frilansprojekt byggt med React, TypeScript och Vite.

## Funktioner

- Snabb utvecklingsmiljö med Vite
- TypeScript för typkontroll
- ESLint och Prettier för kodkvalitet
- WCAG

## Kom igång

```bash
# Installera beroenden
npm install

# Starta utvecklingsserver
npm run dev

# Bygg för produktion
npm run build
```

## Projektstruktur

```
├── public/           # Statisk media, bilder, fonts, Vanta/Three.js
├── src/              # Källkod
│   ├── components/   # React-komponenter
│   ├── App.tsx       # Huvudkomponent
│   ├── App.css       # Global CSS
│   └── main.tsx      # Entrypoint
├── .gitignore        # Git ignore-regler
├── package.json      # Projektmetadata
├── vite.config.ts    # Vite-konfiguration
└── README.md         # Denna fil
```

## Verktyg och plugins

- [Vite](https://vitejs.dev/)
- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [ESLint](https://eslint.org/)
  ESLint och Prettier är installerade för kodkvalitet och formatering. Standardregler används för React och TypeScript.
